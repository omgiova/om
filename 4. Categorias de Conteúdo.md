# 4. Categorias de Conteúdo

---

## Categorias

A voz da Open Mídia se organiza em **4 categorias**. Todo conteúdo produzido se encaixa em uma delas.

| **Educação** | **Bastidores** | **Resultado** | **Branding** |
|---|---|---|---|
| Marketing de conteúdo | Processo de criação de conteúdo | Cases de sucesso com dados | Conceito anti-ads |
| Educação ampla | Primeiros 30 dias de um cliente novo | Depoimentos de clientes | O que a Open Mídia é e não é |
| Educação nichada | Etapas antes de publicar o 1º post | Números e histórico da agência | Orgânico como escolha inteligente |
| Materiais gratuitos | Dia a dia da agência | Resultados de engajamento e alcance | Posicionamento de longo prazo vs. resultado imediato |
| Conteúdo orgânico vs. tráfego pago | Bastidores leves e cotidiano | | Valores da marca |
| SEO, GEO e AEO | Repost e interação com clientes | | |
| IA aplicada ao marketing | | | |
| Dados de mercado e tendências | | | |
| Navegação passiva x navegação ativa | | | |
| Revista nichada por nicho | | | |
